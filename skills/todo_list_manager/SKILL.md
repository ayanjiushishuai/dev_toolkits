# Todo List Manager

管理 todo.md 和 work_done.md 的开发任务生命周期。

## 使用场景

- 用户说"记录这个需求"、"帮我看看优先级"
- 用户说"XX功能完成了"、"生成今日计划"
- 用户说"看看今天该做什么"、"本周计划"、"本月计划"、"进度如何"
- 需要调整任务优先级、分析工作量、验证任务完成状态、整理待办事项

## 自动化触发

- 每周一早上：自动触发周计划确认
- 每月1号：自动触发月计划生成

## 任务状态

- `pending`：待处理
- `in_progress`：进行中
- `completed`：已完成
- `blocked`：被阻塞

## 操作

### 添加任务
```
- [ ] <任务描述> [@<标签>] [#<优先级>]
```

### 完成任务
移动任务从 todo.md 到 work_done.md，并记录完成时间。

### 优先级

- P0：紧急重要
- P1：重要
- P2：一般
- P3：低优先级

## 文件结构

- `todo.md`：待办任务列表
- `work_done.md`：已完成任务记录
